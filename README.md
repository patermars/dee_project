# Smart City Traffic Ingestion

Real-time traffic monitoring system using TomTom API, deployed on Render.com with Railway MySQL.

## Features
- Traffic flow monitoring
- Incident detection
- Automated data collection every minute
- Free deployment on Render.com

## Endpoints
- `/` - Home
- `/health` - Health check
- `/status` - Database status
- `/init-db` - Initialize database
- `/ingest` - Trigger data collection
